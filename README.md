# webcut

브라우저 안에서 돌아가는 가벼운 영상 편집기. **업로드 없음 · 서버 없음** — 영상이 당신의 기기 밖으로 나가지 않습니다.

FFmpeg.wasm로 자르고(trim) 이어붙이고(concat), Final Cut 스타일 타임라인에서 편집합니다.

**라이브 데모:** https://seongilp.github.io/webcut/

## 기능

- Final Cut 스타일 3분할 레이아웃(라이브러리 / 뷰어 / 인스펙터) + 마그네틱 타임라인
- **시퀀스 재생(프로그램 모니터)** — 재생헤드가 클립 경계를 넘어 연속 재생
- **범위 선택** — 타임라인을 드래그해 구간 선택 → 선택 영역만 남기기 / 삭제
- **클립 트리밍 핸들** — 클립 양 끝 드래그로 In/Out 조정
- **블레이드(분할)**, **마커**, **스내핑**, **스키밍**
- **줌 슬라이더 + 맞춤**
- **실행취소 / 다시실행** (⌘Z / ⌘⇧Z)
- **내보내기** — 빠른 복사 컷 / 정밀 모드(재인코딩) → mp4 다운로드

## 키보드

`Space` 재생 · `I`/`O` In/Out · `E` 추가 · `B` 분할 · `M` 마커 · `R` 범위 도구 · `V` 선택 도구 ·
`N` 스냅 · `S` 스킴 · `←/→` 프레임 이동(Shift=1초) · `↑/↓` 편집점 · `Del` 삭제 · `Esc` 범위 해제

## 기술

- 순수 정적 사이트 (빌드 불필요). `@ffmpeg/ffmpeg`·`@ffmpeg/util` ESM은 동봉(같은 출처 Web Worker용), FFmpeg 코어(~25MB)만 최초 1회 CDN에서 로드.
- 단일 스레드 FFmpeg 코어라 cross-origin isolation(COOP/COEP) 불필요 → GitHub Pages에서 그대로 동작.
